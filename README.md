Título: OrganizadorFinanceiro
Status: em desenvolvimento
Tecnologias empregadas: linguagem java e SGBD MySQL
Time de desenvolvedores: Rafael S. Oliveira
Objetivo: Programa desktop que tem por finalidade o controle pessoal de despesas e poupança.
Funcionalidades (requisitos): 
  RF01 - cadastrar usuário com nome, renda mensal, login e senha;
  RF02 - acessar o programa com login e senha do usuário;
  RF03 - atualizar a renda mensal do usuário;
  RF04 - cadastrar e excluir despesas recorrentes do usuário informando descrição, valor do mês atual e valor dos 2 meses 
	anteriores;
  RF05 - cadastrar e excluir despesas não recorrentes do usuário informando descrição, valor mensal e quantidade de meses;
  RF06 - cadastrar ou atualizar o valor mensal a ser poupado e a taxa de remuneração de juros anual a ser aplicada;
  RF07 - mostrar tabela com previsão financeira mensal para os próximos 12 meses contendo despesas recorrentes, despesas não 
  recorrentes, despesa total, poupança e saldo disponível. além disto, deve ser exibida a poupança acumulada nos próximos 12 
  meses.
